🚪 AB 自动门对接配置

AB 门 base URL (POST/GET /acs/door/{doorId})
POST /acs/door/DOOR_OUTER 开/关外门 POST /acs/door/DOOR_INNER 开/关内门 GET /acs/door/{doorId} 查询门状态
📡 区域管控对接配置
ASAP 通过以下三个接口实现风淋区域独占访问
进入区域 POST /api/zones/enter
退出区域 POST /api/zones/exit
状态查询 GET /api/zones/status
区域被占用返回 409，ASAP 自动重试 进入成功获得 permission_id 退出失败自动重试直到释放
